Hyprland Dotfiles

These are my personal dotfiles and Hyprland configs. They include everything required after a minimal Arch Linux install, but they can be used on other distros as well.

The setup is clean, minimal and focuses on a simple workflow.

After downloaded the repo use "restore-dotfiles.sh" to copy the files to their locations. When finished restart hyprland.
If you modify config files and wish push it to github use "collect-dotfiles.sh" to copy all config files to ~/dotfiles.
