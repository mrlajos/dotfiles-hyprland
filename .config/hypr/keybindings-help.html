Hyprland Keybindings

Quick reference for your current setup: windows, workspaces, apps, and system controls.

Windows & Layout

  • SUPER+Q
    Close active window
  • SUPER+F11
    Toggle fullscreen
  • SUPER+SPACE
    Toggle floating mode
  • SUPER+Mouse Left
    Move window
  • SUPER+Mouse Right
    Resize window
  • SUPER+ ← / → / ↑ / ↓
    Change focus between windows

Workspaces

  • SUPER+1..5
    Switch to workspace 1–5
  • SUPER+SHIFT +1..5
    Move window to workspace 1–5
  • SUPER+Mouse Scroll Up/Down
    Scroll between workspaces

Apps

  • SUPER+A
    App search (wofi)
  • SUPER+ENTER
    Open terminal (kitty)
  • SUPER+B
    Open browser (firefox)
  • SUPER+F
    Open file manager (nemo)
  • SUPER+V
    Open clipboard history
  • SUPER+PRINT
    Take a screenshot (select region)

System

  • SUPER+L
    Lock screen
  • SUPER+P
    Power menu
  • SUPER+SHIFT +Q
    Exit Hyprland
  • SUPER+SHIFT +R
    Restart Hyprland
  • SUPER+SHIFT +W
    Restart Waybar
  • SUPER+F1
    Show keybindings help
